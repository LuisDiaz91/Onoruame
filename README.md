# Onoruame

Sistema de gestión de rutas para el PJCDMX. Soporte para 50+ repartidores concurrentes.

## Stack
- Python/Flask
- PostgreSQL
- Redis
- Celery
- Telegram Bot
- Docker/Railway

git clone https://github.com/LuisDiaz91/Onoruame
